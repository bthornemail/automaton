## Generation and Maintenance

### Source Files

The wiki is generated from source documentation using:

- `extract-frontmatter-trie.ts` - Extract frontmatter from docs
- `build-reference-trie.ts` - Build reference hierarchy
- `generate-wikipedia-docs.ts` - Generate Wikipedia-style articles
- `generate-arxiv-paper.ts` - Generate LaTeX paper
- `generate-index.ts` - Generate indexes
- `generate-wiki.ts` - Main orchestrator

### Data Files

- `frontmatter-trie.json` - Extracted knowledge (1.3 MB)
- `academic-citations.json` - Wikipedia/arXiv citations
- `reference-trie.json` - Hierarchical references

### Regeneration

To regenerate the entire wiki:

```bash
cd /home/main/automaton
tsx wiki/generate-wiki.ts
```
