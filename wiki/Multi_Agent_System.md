## üéº How Agents Coordinate: The Orchestra in Action

### The Blackboard: The Shared Score

**What is the blackboard?** A shared knowledge base where agents read and write.

**Why blackboard?** Because agents need to coordinate. The blackboard enables coordination.

**The metaphor**: Like a shared score in an orchestra. All musicians read from it. All write to it.

**The story**: Early CTC had agents that couldn't coordinate. The blackboard emerged from needing coordination. It became essential.

**How it works**:
1. **Agents write facts**: "I discovered this"
2. **Agents read facts**: "What did others discover?"
3. **Agents coordinate**: "Based on what I read, I'll do this"

**The insight**: Coordination emerges from the blackboard. Agents don't need to know about each other‚Äîthey coordinate through the blackboard.

### Communication Patterns: How Agents Talk

**How do agents communicate?** Through the blackboard, not directly.

**Why indirect?** Because indirect communication enables loose coupling. Agents don't need to know about each other.

**The metaphor**: Like musicians in an orchestra. They don't talk directly‚Äîthey watch the conductor (blackboard) and listen to each other.

**The story**: Early CTC had direct agent communication. Indirect communication emerged from needing loose coupling. It became essential.

**Communication patterns**:
- **Write-Read**: Agent writes, others read
- **Subscribe-Notify**: Agent subscribes, gets notified
- **Query-Response**: Agent queries, gets response

**The insight**: Indirect communication enables loose coupling. Agents coordinate without tight coupling.
